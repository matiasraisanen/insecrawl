# Insecrawl

Insecam Crawler

Crawls insecam pages on your command, and downloads a still frame from cameras.

## Run

```
$ python3 insecrawl.py
```

The program is interactive, and will guide you forward.

## Project structure

```
.

│   └── README.md
├── insecrawl.py        # Main program
├── LICENSE
```
